# BinaryCSS

## Links:
- Docs: https://doc-binarycss.netlify.app
- Demo1: https://binarycss-live-demo-1.netlify.app/
- Demo2: https://binarycss-live-demo-2.netlify.app/

### Features:
- reset
- container (responsive)
- font size, weight and alignment
- borders
- height and width
- margin and padding
- font color and background color
- media queries
- Components:
    - Button
    - Card
    - Alert
    - Badge
    - Form
    - Image
    - Navbar
    - Table
    - Modal
    - Dropdown
    - Accordion
    - Carousel
    - Tooltip
- Grid
- Flex
- Arbitary Classes(m-[10px], py-[1rem])
    - margin
    - padding
    - height
    - weight
    - font-size
    - text-color (tc-[])
    - background-color (bg-[])
